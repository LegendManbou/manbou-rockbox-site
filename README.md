# Manbou / RockBox

Manbou / RockBox の自己紹介サイトです。AI、創作、学習、発信を少しずつ形にしていく大学1年生の活動ハブとして作っています。

## サイト概要

趣味側の活動と、AI・学習・仕事に向けた活動を1つのサイト内で見やすく分けて紹介します。
完成済みのポートフォリオではなく、これから作るものや学習記録を積み上げていくための場所です。

## 使用技術

- HTML
- CSS
- JavaScript

外部ライブラリは使わず、静的サイトとして作成しています。

## アクセス解析について

Google Analyticsを使って、閲覧数やクリックなどの集計データを確認しています。
個人を特定する目的ではなく、サイト改善のために使います。

## ファイル構成

```text
.
├── index.html

```

## 公開方法

GitHub Pagesで公開する場合の基本手順です。

1. このフォルダをGitHubリポジトリにアップロードする
2. GitHubのリポジトリ設定から Pages を開く
3. 公開元を `main` ブランチのルートに設定する
4. 発行されたURLで `index.html` が表示されることを確認する
